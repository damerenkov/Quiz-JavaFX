Main Form
1. При запуске программы появится главное окно в котором можно выбрать два варианта загрузки викторины:
	а) "From Internet" - если программа запускается впервые 
	б) "From File" - если вариант викторины был сохранен ранее
а также возможность показать правильные ответы в конце викторины.

Loading Form
2. В окне Loading Form необходимо сконфигурировать будущую викториню:
	а) Поле "Number of questions from 1 to 10" - в данном поле необходимо указать количество вопросов.
	б) Список "Select Category" - необходимо выбрать одну из указанных тем викторины.
	в) Список "Select Difficult" - необходимо выбрать сложность викторины.
* ПРИМЕЧАНИЕ (Категория знаменитости на уровне сложности Hаrd имеет только 5 вопросов)

Кнопка Save - есть возможность сохранить данную конфигурацию викторины на компьютере пользователя для последующей локальной игры, которую можно будет загрузить в главном окне при старте программы

Кнопка Start - запускает текущую конфигурацию викторины.

3. В окне Game Form нужно ответить на все вопросы и после перейи во вкладку Result и нажать на кнопку Check для вывода результата.
